# TMH16 Alignment Layer

The rules engine checks evidence completeness status:
- complete
- incomplete
- review_required
- not_observed
- not_applicable

## Checks
- Peak 15-minute movement data available
- Separate movement reporting by turn/stream
- Queue evidence exported
- Pedestrian observations included
- Access/school/parking/public transport/service observations captured where relevant

> This is a checklist support layer. It is **not** legal compliance or certification.
